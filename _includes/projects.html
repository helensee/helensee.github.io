Steam Wish

Use my account: kuroiblood
About: The web app pulls the users wishlist and has a place for the user to sort through it based on title, sale price, and/or tag. First page lists the top 50 and has a bar graph of the games original price and sale price. Last page is a small recommendation page. It gives a few recommended games based on the top 3 tags of the user.
I focused on working with the data.

Technology: MySQL databse, Amazon Web Services, Node.js, Express.js, D3.js, and Bootstrap

I focused on the MySQL database. I pulled data for my teammates with queries. I also worked with D3.js to build a bar graph to compare sale prices with the orgiginal prices.
